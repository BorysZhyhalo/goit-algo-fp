# Домашнє завдання 2 — Дерево Піфагора

## Опис

Рекурсивна побудова фрактала «дерево Піфагора» з візуалізацією через `turtle`. Користувач задає рівень рекурсії при запуску програми. макстимально допустимий рівень - до 10. 

## Файли

| Файл | Призначення |
|------|-------------|
| `main.py` | Рекурсивне малювання фрактала та введення рівня |

## Запуск

```bash
cd pythagoras_tree
python main.py
```

Після запуску введіть рівень рекурсії, наприклад `3`.

## Алгоритм

1. Малюється квадрат (сторона `size`).
2. На верхній стороні будуються дві гілки під кутами ±45°.
3. Сторона дочірнього квадрата: `size / sqrt(2)`.
4. Рекурсія зупиняється, коли `level == 0`.

## Параметри

| Рівень | Що малюється |
|--------|--------------|
| 0 | 1 квадрат |
| 1 | 1 + 2 квадрати |
| 2 | 7 квадратів |
| 3+ | все більше гілок |


## Висновки

1. Фрактал побудовано через **рекурсію**: кожен квадрат породжує 2 нові гілки.
2. Розмір гілок зменшується пропорційно `1/√2`, тому дерево не розходиться нескінченно.
3. `turtle` дозволяє наочно побачити геометрію поворотів на ±45°.
4. Обмеження рівня рекурсії захищає від надто довгого малювання та `RecursionError`.
