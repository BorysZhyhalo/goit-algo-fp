# Домашнє завдання 5 — Візуалізація обходу бінарного дерева

## Опис

Програма візуалізує обходи бінарного дерева **у глибину (DFS)** та **у ширину (BFS)**. Кожен крок обходу показується окремим кадром: відвідані вузли отримують унікальний колір у форматі RGB hex (`#0A3D7A` → `#A8D4F5`).

## Файли

| Файл | Призначення |
|------|-------------|
| `main.py` | DFS (стек), BFS (черга), покрокова візуалізація |


## Алгоритми

### DFS — стек (без рекурсії)

```text
stack = [root]
while stack:
    node = stack.pop()
    visit(node)
    push right, then left
```

Порядок для демо-дерева: `0 -> 4 -> 5 -> 10 -> 1 -> 3`

### BFS — черга (без рекурсії)

```text
queue = [root]
while queue:
    node = queue.popleft()
    visit(node)
    enqueue left, then right
```

Порядок для демо-дерева: `0 -> 4 -> 1 -> 5 -> 10 -> 3`

## Кольори

| Параметр | Значення |
|----------|----------|
| Невідвідані вузли | `#E0E0E0` |
| Перший відвіданий | `#0A3D7A` (темний) |
| Останній відвіданий | `#A8D4F5` (світлий) |
| Формат | 16-ричний RGB hex |

Колір залежить від порядкового номера відвідування: чим пізніше вузол — тим світліший відтінок.

## Структура рішення

1. `build_demo_tree()` — дерево
2. `dfs_iterative()` — обхід у глибину через `stack`
3. `bfs_iterative()` — обхід у ширину через `deque`
4. `visit_color()` — градієнт hex-кольорів
5. `visualize_traversal()` — покрокове малювання

## Висновки

1. **DFS** використовує стек (LIFO): спочатку йде в глибину однієї гілки.
2. **BFS** використовує чергу (FIFO): спочатку обходить вузли рівня за рівнем.
3. Кольори відображають **порядок відвідування** від темного до світлого.
