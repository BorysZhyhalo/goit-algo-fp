# Домашнє завдання 3— Алгоритм Дейкстри

## Опис

Реалізація алгоритму Дейкстри для знаходження найкоротших шляхів у зваженому графі. Для оптимізації вибору вершини використовується **бінарна купа** (`heapq`).

## Файли

| Файл | Призначення |
|------|-------------|
| `main.py` | Клас `Graph`, функція `dijkstra()` та демонстрація |

## Запуск

```bash
cd dijkstra
python main.py
```

## Структура рішення

| Компонент | Опис |
|-----------|------|
| `Graph` | Зважений граф у вигляді списку суміжності |
| `add_vertex()` | Додавання вершини |
| `add_edge()` | Додавання орієнтованого ребра з вагою |
| `dijkstra()` | Пошук найкоротших шляхів від стартової вершини |
| `heapq` | Мін-купа для вибору найближчої вершини |

## Алгоритм Дейкстри

1. Ініціалізувати відстані: для старту `0`, для інших — `∞`.
2. Додати стартову вершину в мін-купу `(відстань, вершина)`.
3. Поки купа не порожня:
   - взяти вершину з мінімальною відстанню;
   - для кожного сусіда оновити відстань, якщо знайдено коротший шлях;
   - додати сусіда в купу.
4. Повернути словник найкоротших відстаней.

## Приклад виводу

```text
Граф (орієнтований, зважений):
  A -> B(4), C(2)
  B -> C(1), D(5)
  C -> D(8), E(10)
  D -> E(2), F(6)
  E -> F(3)
  F ->

Найкоротші шляхи від вершини 'A':
  A -> A: 0
  A -> B: 4
  A -> C: 2
  A -> D: 9
  A -> E: 11
  A -> F: 14
```

## Складність

| Операція | Складність |
|----------|------------|
| Dijkstra з бінарною купою | O((V + E) log V) |
| Dijkstra без купи | O(V²) |

де `V` — кількість вершин, `E` — кількість ребер.

## Висновки

1. **Алгоритм Дейкстри** коректно знаходить найкоротші шляхи у зваженому графі без від'ємних ребер.
2. **Бінарна купа** прискорює вибір найближчої вершини порівняно з лінійним пошуком.
3. **Список суміжності** зручний для представлення розріджених графів.
4. Результат — відстані від початкової вершини до всіх інших (або `∞`, якщо вершина недосяжна).
