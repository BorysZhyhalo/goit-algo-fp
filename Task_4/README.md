# Домашнє завдання 4 — Візуалізація бінарної купи

## Опис

На базі коду для візуалізації бінарного дерева реалізовано функцію `draw_heap()`, яка будує дерево з масиву бінарної купи та відображає його у вигляді піраміди.

## Файли

| Файл | Призначення |
|------|-------------|
| `main.py` | Побудова дерева з купи та візуалізація |

## Запуск

```bash
cd heap_visualization
python main.py
```

Потрібні бібліотеки:

```bash
pip install networkx matplotlib
```

## Нові функції

| Функція | Опис |
|---------|------|
| `build_heap_tree(heap)` | Рекурсивно будує `Node`-дерево з масиву купи |
| `draw_heap(heap)` | Будує дерево і викликає візуалізацію |

## Як масив перетворюється на дерево

Для індексу `i` у масиві (0-indexed):

```text
лівий  = 2 * i + 1
правий = 2 * i + 2
```

**Приклад:** `[15, 10, 8, 5, 3, 1]`

```text
       15
      /  \
    10    8
   / \   /
  5  3  1
```

## Структура рішення

1. `build_heap_tree()` — створення дерева з купи
2. `add_edges()` — побудова графа та координат (з базового коду)
3. `draw_tree()` — малювання через NetworkX і Matplotlib

## Висновки

1. Масив бінарної купи можна перетворити на бінарне дерево за індексами нащадків.
2. `draw_heap()` об'єднує побудову дерева та його відображення в одну функцію.
