# Домашнє завдання 7 — Метод Монте-Карло (кидання кубиків)

## Опис

Симуляція великої кількості кидків двох кубиків методом Монте-Карло. Програма обчислює ймовірність кожної суми (2–12), виводить таблицю порівняння з аналітичними значеннями та будує графік.

## Файли

| Файл | Призначення |
|------|-------------|
| `main.py` | Симуляція, таблиця ймовірностей, графік |

## Запуск

```bash
cd dice_monte_carlo
pip install matplotlib
python main.py
```

## Алгоритм

1. Виконати `N` кидків двох кубиків (`random.randint(1, 6)`).
2. Для кожного кидка порахувати суму.
3. Підрахувати частоту кожної суми від 2 до 12.
4. Обчислити ймовірність: `P(сума) = count / N`.
5. Порівняти з аналітичними значеннями `k/36`.

## Аналітичні ймовірності

| Сума | Ймовірність |
|------|-------------|
| 2 | 1/36 ≈ 2.78% |
| 3 | 2/36 ≈ 5.56% |
| 4 | 3/36 ≈ 8.33% |
| 5 | 4/36 ≈ 11.11% |
| 6 | 5/36 ≈ 13.89% |
| 7 | 6/36 ≈ 16.67% |
| 8 | 5/36 ≈ 13.89% |
| 9 | 4/36 ≈ 11.11% |
| 10 | 3/36 ≈ 8.33% |
| 11 | 2/36 ≈ 5.56% |
| 12 | 1/36 ≈ 2.78% |

## Приклад виводу таблиці

```text
Сума    Monte Carlo       Теорія      Різниця
----------------------------------------------
2            2.79%        2.78%       +0.01%
3            5.54%        5.56%       -0.02%
...
7           16.65%       16.67%       -0.02%
...
12           2.80%        2.78%       +0.02%
```

## Висновки

1. **Метод Монте-Карло** дає наближення до теоретичних ймовірностей через частоту подій.
2. При `N = 100 000` відхилення від аналітичних значень зазвичай менше 0.1%.
3. Розподіл **симетричний**, найбільша ймовірність — для суми **7**.
